Templates for initializing project structure.

Install packages for chosen rendering script:
  
  (ruby) gem install [--user-install] thor

Example usage:
  
  thor install template_cli.thor ; thor installed
  
  thor template:cli [--data data.yaml] templatethor-rb [--kvhash date:2013-10-01]
  
  ruby render_eruby.rb [--data data.yaml] skeleton-rb [date=2013-10-01]
